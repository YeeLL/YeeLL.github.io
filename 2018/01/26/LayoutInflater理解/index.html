LayoutInflater理解

LayoutInflater是我们用来解析XML布局文件的类,创建它有3种方式:

  • LayoutInflater.from(this)
  • getLayoutInflater （Activity提供的方法）
  • this.getSystemService(Context.LAYOUT_INFLATER_SERVICE)

这三种方法本质上都是调用
this.getSystemService(Context.LAYOUT_INFLATER_SERVICE);

解析布局，生成View

解析布局生成View也有两种方式

  • inflate(@LayoutRes int resource, @Nullable ViewGroup root, boolean attachToRoot)
  • inflate(@LayoutRes int resource, @Nullable ViewGroup root)

实际上第二种最后也是调用的
inflate(@LayoutRes int resource, @Nullable ViewGroup root, boolean attachToRoot)

inflate(@LayoutRes int resource, @Nullable ViewGroup root, boolean attachToRoot)

inflate(@LayoutRes int resource, @Nullable ViewGroup root)

inflate(XmlPullParser parser, @Nullable ViewGroup root, boolean attachToRoot) 最终的xml解析并生成view的方法

自此xml就生成了view 最主要的还是XML的解析。

补充

Activity 的setContentView方法 如果传入的是layout ID 那么最终就会到PhoneWindow的setContentView方法中去执行mLayoutInflater.inflate(id,mParentView)
